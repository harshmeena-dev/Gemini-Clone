# Gemini Clone

This project is a clone of the Gemini cryptocurrency exchange, built to provide users with functionalities similar to the official Gemini platform.

## Setup Instructions

### Prerequisites
- Make sure you have Python 3.8 or higher installed on your machine.

### 1. Create a `.env` File
Create a file named `.env` in the root directory of the project and add your Gemini API key:
API_KEY=your_api_key_here


### 2. Set Up a Virtual Environment     
To set up a virtual environment, run the following commands:
   
**Navigate to your project directory**   
cd path/to/Gemini-Clone    
   
**Create a virtual environment**   
python -m venv env   

**Activate the virtual environment**   
**On Windows:**   
.\env\Scripts\activate   
**On macOS/Linux:**   
source env/bin/activate   

### 3. Install Requirements
Install the project dependencies listed in requirements.txt by executing:
python -m pip install -r requirements.txt

### 4. Run the Application
After the setup type the command to run the python file:
python app.py
